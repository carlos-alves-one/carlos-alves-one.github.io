    • The website must be operable
  • Accessibility aspect 3
    • The website must be understandable

How have you made the site easy to navigate?

Navigation on my website will at the bottom of the page. So that navigation is carried out a direct and straightforward way. All the pages' content will be present in a simplified and summarized form, with connection to other pages to present more or complement previously presented content. The entire layout designed so that a person with poor eyesight can access the content easily. The colours were selected so as not to disturb the correct visualization of the pages and their content. Finally, another decision made to eliminate any photo or image so as not to conflict with copyright.

Implementation

Learning

The website development would never be possible before there were first learning and practice of the general knowledge of HTML, CSS and JavaScript. And without a doubt, the lectures presented complemented with other information, namely online and reading books related to the subject, were undoubtedly fundamental so that it was possible to present my website as the final result of all the acquired learning.

Libraries and other resources

In this project, I did not use any library, for the fact to present in more detail all the implemented code and to be able to control better the final result, namely the use of CSS. I will undoubtedly implement the use of libraries to present better design and functionality.

Evaluation

What worked well

I can say that the final result of the site was what I expected to be. It was not an easy process, but the final result, in my opinion I think was very positive. I liked the part where I developed and integrated the numbers game with the rest of the site. The implementation of this game and the rest of the code allowed me to present my knowledge of JavaScript acquired in the last term with HTML and CSS.

What might be improved

Undoubtedly can improve the use and implementation of libraries and a better responsive website.
